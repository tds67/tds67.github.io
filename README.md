# Cybersecurity in the Kurdistan Region

A lightweight, responsive static website about the importance of cybersecurity in the Kurdistan Region:
- Advancements
- Current weaknesses
- Practical roadmap
- Sources & further reading

## How to run
### Option A (quick)
Open `index.html` in a modern browser.

### Option B (recommended)
From this folder, run:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Structure
- `index.html` – overview
- `advancements.html`
- `weaknesses.html`
- `roadmap.html`
- `resources.html` – references + license
- `assets/` – CSS/JS/images
